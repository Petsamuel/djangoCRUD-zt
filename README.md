## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/{{Petsamuel/djangoCRUD-zt}}

# Access
$ cd bieefilled

# Install dependencies
$ pip install django

# Run the project
$ python3 manage.py runserver

# The server will initialize in the <http://localhost:3000>
```
